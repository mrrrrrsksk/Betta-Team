# DomEventsAsyncJs

Bu layihə sadə bir **Kalkulyator**-dur. 

## Funksiyalar:
- Toplama, çıxma, vurma, bölmə, faiz
- AC (hamısını sil), CE (sonuncunu sil)
- `eval()` ilə nəticə hesablanması
- DOM Events ilə idarə olunur
- `setTimeout`, `setInterval` istifadə olunub
- `async/await` və `fetch API` ilə nümunə göstərilib

## Texnologiyalar:
- HTML
- CSS
- JavaScript (DOM, Events, Async/Await, Fetch API, Window obyektləri)
